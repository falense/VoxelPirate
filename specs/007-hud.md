# Spec 007 — Heads-up display

**Status:** accepted (2026-07-02)

## Summary

Replace the four rows of top-left text with a proper HUD built from Bevy
UI nodes:

- **Status panel** (top-left, translucent rounded panel): gold title
  `Class | Wave N` ("Wave N next" at the dock), a salvage/kills line, and
  three gauges — HULL (fill color runs green → red with damage) and
  PORT/STBD broadside bars that fill as the guns reload and turn gold
  when ready. Gauges replace the old `Hull 87% / Port 1.3s` text.
- **Intel line** (under the panel): the lookout's wind/foe/derelict/
  flotsam bearings, unchanged in content.
- **Center announcements**: unchanged (wave banners, pause, sinking,
  victory).
- **Key hint** (bottom-right, dim): contextual — dock actions at the
  dock, helm keys under sail, build keys in battle build mode. Replaces
  the old always-on controls line.
- **Block hotbar** (bottom-center, visible whenever build mode is live):
  ten slots mapped to digits 1-0, each showing its block's actual
  procedural texture tile as the icon (`GameAssets.block_tiles`, kept
  main-world resident for UI use), digit badge in the corner, gold border
  on the selected slot, and a `Name | cost` label above the bar.

HUD elements the updater writes each frame are found by role enums
(`HudText`, `HudBar`) — one query each, no marker-component tangle.

## Font gotcha

Bevy's default font has no `·`, `—`, or `☠` glyphs; they render as tofu
boxes (they always had — the old HUD was full of them). All UI strings and
announcements now stick to ASCII separators (`|`, `:`). If a richer
glyph set is ever wanted, ship a font asset.

## Verification

`--selftest` screenshots: dock shows the panel (full gauges), hotbar with
textured icons and gold slot-1 selection, dock hint; battle shows the
starboard gauge draining after the broadside, intel bearings with foe
range, and the sail hint. Self-test values unchanged (scrap 1 → place 0).
